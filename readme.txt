== Description ==

Shows all attachments for a post or page in a box on the edit page. Supports all custom post types.

== Installation ==

1. Upload directory `see-attachments` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Visit the edit post or edit page to see the boxes with info.
